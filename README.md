# TDD_in_Rspec

> A project that involves implementing a class with some methods, but started off by doing a test-driven development (TDD)

## Built With

- Ruby
- TDD
- Rspec
- Rubocop

## Getting Started

### Clone this repository

```
bash
$ git clone git@github.com:Favourezeugwa/TDD_in_Rspec.git
$ cd TDD_in_Rspec
```

### Run project

```
vs-code terminal
$ Rspec spec
```
